# 상품 이미지 가이드

## 사진이 안 나올 때

- 현재는 **로컬 placeholder** (`/images/placeholder.svg`)를 쓰고 있어서 외부 링크 없이 표시됩니다.
- 실제 상품 사진으로 바꿀 때는 아래 규칙만 지키면 됩니다.

---

## 이미지 어디서 구하나요?

1. **직접 촬영** – 실제 판매하는 제품을 촬영 (가장 신뢰도 높음)
2. **제조사/공급처** – 정식 제공하는 제품 이미지 사용 (저작권 확인)
3. **스톡/무료 이미지** – 상품과 맞는 무료 이미지 (Unsplash, Pexels 등) — Tesla/자동차 부품 검색

---

## 권장 사이즈

| 용도 | 권장 크기 | 비고 |
|------|-----------|------|
| **리스트/카드 썸네일** | **800×800 px** (정사각형) | 한 번 이 크기면 대부분 화면에서 충분 |
| **상세 페이지 메인** | **1200×1200 px** | 더 선명하게 보이게 할 때 |
| **상세 갤러리 추가 이미지** | 800×800 ~ 1200×1200 | 메인과 같은 비율 권장 |

- **비율**: **1:1 (정사각형)** 권장 — 그리드/카드 레이아웃이 깔끔해짐.
- **파일 형식**: JPG(사진), PNG(로고·투명 필요할 때), WebP(용량 줄일 때).

---

## 배경색 상관있나요?

- **상관있습니다.** 배경을 통일하면 브랜드 느낌이 좋아집니다.
- **추천**:
  - **흰색 (#ffffff)** 또는 **밝은 회색 (#f5f5f5)** — 대부분 이커머스에서 쓰는 방식.
  - Tesla/Apple 느낌을 살리려면 화이트 또는 아주 연한 그레이가 잘 맞습니다.
- **피하는 것**: 제품마다 배경색이 제각각이면 리스트에서 어수선해 보입니다.

---

## 홈페이지에서 쓰는 파일 이름 (필수)

이미지는 `public/images/` 안에 **아래 이름 그대로** 두면 됩니다. (확장자는 `.jpg` 또는 `.png` 둘 다 가능)

| 상품 | 메인 이미지 | 갤러리 추가 이미지 |
|------|-------------|---------------------|
| Cabin Air Filter | `cabin-filter.jpg` | `cabin-filter-2.jpg`, `cabin-filter-3.jpg` |
| Wiper Blade Set | `wiper-set.jpg` | `wiper-set-2.jpg` |
| Key Card | `key-card.jpg` | `key-card-2.jpg` |
| Interior Cleaning Kit | `cleaning-kit.jpg` | `cleaning-kit-2.jpg`, `cleaning-kit-3.jpg` |

다운로드한 파일 이름이 다르면 `scripts/rename-images.sh` 를 편집해서 실행하거나, 직접 위 이름으로 바꿔서 `public/images/`에 넣으면 됩니다.
